# xiloader

## Build

```sh
mkdir build
cd build
cmake .. -A Win32
cmake --build .
```

## xi_checker

Verifies DirectPlay and POL are installed and connect to a server using xiloader (installed in POL folder).

Usage:

```sh
xi_checker $server_ip
```
